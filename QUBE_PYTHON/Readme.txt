1. Download Visual Studio Code if you do not already have it (optional, but recommended).

2. Make sure python is correctly installed on your device.

3. Open vscode and go to extensions. Search for "Python" and install it.

4. Open the QUBE folder by pressing file -> open folder -> .../QUBE

5. Install the required libraries by running "install.py" or copying and running: "pip install pyserial PyQt5 pyqtgraph"
   in a terminal.

6. To test that everything is working with the QUBE run "gui.py". Make sure to set the COM_PORT in control.py.

7. Run main.py to run the main program. Write your code in "control.py".
